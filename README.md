# 🚀 Customer Churn Prediction System

## 📌 Overview

This project is an end-to-end Machine Learning application that predicts whether a customer is likely to churn (leave a service) based on their subscription details.

The system includes data preprocessing, model training, evaluation, and deployment using an interactive web interface.

---

## 🎯 Features

* Predict customer churn (Yes/No)
* Display probability of churn
* User-friendly web interface using Streamlit
* Real-time predictions based on user input
* Clean and interpretable output

---

## 🧠 Machine Learning Workflow

1. Data Cleaning & Preprocessing
2. Feature Selection
3. Model Training (Logistic Regression, Random Forest)
4. Model Evaluation (Accuracy, Confusion Matrix)
5. Model Selection (Random Forest)
6. Deployment using Streamlit

---

## 📊 Model Performance

* Logistic Regression Accuracy: ~90%
* Random Forest Accuracy: ~92%
* Final Model Used: **Random Forest Classifier**

## ▶️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/customer-churn-prediction.git
cd customer-churn-prediction
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```

4. Open in browser:

```
http://localhost:8501
```
